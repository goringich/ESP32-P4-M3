# mpu9250

Файл: `components/mpu9250/src/mpu9250.c`.

Назначение:

- проверить адреса `0x68` и `0x69`;
- прочитать регистр `WHO_AM_I`;
- вернуть человекочитаемое имя модели по WHO_AM_I.

Это не полный драйвер MPU9250. Это минимальный слой для обнаружения и базовой диагностики.
