# app_net_tick

Исходник: `components/app/src/app_net.c`.

`app_net_tick()` рассылает WebSocket telemetry клиентам.

Что делает:

- если server не запущен, сразу выходит;
- проверяет, прошла ли секунда с последней рассылки;
- получает список клиентов через `httpd_get_client_list()`;
- строит JSON через `app_net_build_json()`;
- для каждого клиента проверяет `httpd_ws_get_fd_info()`;
- если клиент является WebSocket-клиентом, ставит отправку в очередь через `app_net_queue_ws_send()`.

Почему отправка через queue:

- ESP HTTP server ожидает, что асинхронная отправка будет выполняться через `httpd_queue_work`;
- это безопаснее, чем отправлять кадр прямо из произвольного контекста приложения.
